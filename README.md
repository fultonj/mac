# mac

I primarily use Fedora on a Thinkpad and document its configuration at:

  https://github.com/fultonj/tools/
  
I also use a Mac sometimes and document its configuration in this
repo. Unfortunately, there is some overlap in my tmux configuration.

I use the following to make my mac feel feel more like
[my fedora configuration](https://github.com/fultonj/tools/blob/main/fedora.dm)

- [ghostty](https://ghostty.org/docs/about): terminal
- [karabiner-elements](https://karabiner-elements.pqrs.org): keyboard customizer
- [amethyst](https://ianyh.com/amethyst): tiling window manager
- [hotkey](https://hotkey.macupdate.com): so `command+return` opens a terminal
- [brew](https://brew.sh): package manager (used to install tmux and other packages)

I keep the dot files for some of the above in this repo.

- [ghostty](ghostty)
- [karabiner-elements](karabiner)
- [tmux](tmux) - derivative work of [tools/tmux](https://github.com/fultonj/tools/blob/main/tmux.md)
