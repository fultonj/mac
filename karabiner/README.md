# karabiner

Configuration for [Karabiner-Elements](https://karabiner-elements.pqrs.org).

Copy each file to its destination:

| File | Destination |
|---|---|
| [karabiner.json](karabiner.json) | `~/.config/karabiner/karabiner.json` |

See [ctrld.md](ctrld.md) for how the emacs-remap rules interact with Ghostty's
`ctrl+d` keybind (the Ctrl+D-closes-surface behavior is controlled here, not in
Ghostty).
